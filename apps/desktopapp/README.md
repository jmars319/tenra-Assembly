# tenra Assembly Desktop App

Tauri desktop workbench for tenra Assembly. This app currently provides:

- Local manual content drafts
- Reusable templates for each shared content type
- Promotion checks through shared Assembly validators
- Draft, ready, approved, and rejected review state
- Markdown copy/export for handoff
- One Rust command roundtrip into the frontend
- Consumption of shared workspace packages for types, domain rules, and prompt guidance

Run it from the repo root:

```bash
pnpm run dev:desktop
```
